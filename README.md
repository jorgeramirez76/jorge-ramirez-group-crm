# Jorge Ramirez Group CRM

AI-Powered Real Estate CRM built for The Jorge Ramirez Group at Keller Williams Premier Properties.

## Overview

Self-hosted CRM system replacing GoHighLevel ($297/mo → ~$38/mo). Features:

- **AI Follow-Up Engine** — Automated lead nurturing using Gemma 4 31B (local AI)
- **Voice Agent** — AI cold calling with voice cloning (LiveKit + Cartesia/GPT-SoVITS)
- **Unified Inbox** — SMS, Email, iMessage, Web Chat (Chatwoot)
- **Pipeline Management** — Drag-and-drop Kanban board (Twenty CRM)
- **Email Campaigns** — 45+ email templates following Mulrenin Reverse Selling + NEPQ
- **Automation Engine** — Visual workflow builder (n8n)
- **AI Newsletter** — Weekly market updates generated by AI
- **AI Review Responder** — Auto-responds to Google/Facebook/Zillow reviews
- **AI Home Value Reports** — Monthly personalized reports
- **Personalized Video** — AI avatar videos via Remotion + MagiHuman
- **E-Signature** — Built-in contract management
- **Calendar/Booking** — Appointment scheduling with custom forms

## Tech Stack

| Component | Technology |
|---|---|
| CRM | Twenty CRM (Docker) |
| Automations | n8n (Docker) |
| Inbox | Chatwoot (Docker) |
| Database | PostgreSQL + Redis (Docker) |
| AI | Ollama + Gemma 4 31B (local) |
| Dashboard | Next.js + shadcn/ui |
| SMS | Twilio |
| Email | Resend / Amazon SES |
| iMessage | Native macOS AppleScript |
| Voice | LiveKit + Cartesia TTS + Deepgram STT |
| Video | Remotion + daVinci-MagiHuman |
